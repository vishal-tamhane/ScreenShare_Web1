
## 📱 Usage

1. **Create a Room:**
   - Click "Generate Code"
   - Enter your name
   - Share the generated room ID

2. **Join a Room:**
   - Click "Join Room"
   - Enter the room ID
   - Start viewing the shared screen

## 🔐 Security Features

- Unique room ID generation
- Session timeout after 30 minutes
- Rate limiting for room creation
- End-to-end encrypted connections

## 🌐 Browser Support

- Chrome (recommended)
- Firefox
- Edge
- Safari
